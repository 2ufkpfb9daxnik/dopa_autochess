## まとめ：最小限の「全リスト」

かなり圧縮すると、オートチェス系で変更対象になり得るステータスは以下です。

* **HP系**：最大HP、現在HP、HP回復、被回復量、シールド、シールド量
* **防御系**：Armor、Magic Resist、Durability、ダメージ軽減、ブロック、回避、CC耐性
* **攻撃系**：AD、最小/最大ダメージ、AS、BAT、DPS、射程、クリ率、クリダメ、通常攻撃追加効果
* **スキル系**：AP、スキルダメージ、スキル回復量、スキルシールド量、AoE、対象数、効果時間
* **マナ系**：初期マナ、現在マナ、最大マナ、マナ獲得量、マナ回復、マナコスト
* **吸収系**：Lifesteal、Omnivamp、Spell Vamp
* **移動系**：移動速度、突進距離、跳躍距離、ノックバック/引き寄せ距離
* **デバフ系**：Armor低下、MR低下、被ダメ増加、回復阻害、攻撃速度低下、沈黙、スタン、武装解除など
* **召喚/変身系**：召喚数、召喚物ステータス、分身数、分身倍率、変身後ステータス
* **確率/時間系**：発動率、内部CD、持続時間、スタック数、スタック上限
* **盤面/経済系**：プレイヤーHP、レベル、出撃上限、ゴールド、ショップ確率、プール数、特性段階

率直に言うと、「完全に全て」をゲーム横断で固定リスト化するなら、**“ユニットの基礎ステータス”だけでなく、“一時バフ・デバフ・確率・持続時間・リソース・盤面経済”まで含める必要があります**。TFT寄りに限定するなら、主要な基礎ステータスは **HP / AD / AS / Armor / MR / Mana / Starting Mana / Range / AP / Crit / Omnivamp / Damage Amp / Durability** あたりが中心です。
