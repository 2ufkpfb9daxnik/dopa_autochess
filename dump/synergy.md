以下は「**TFTで何度も形を変えて出てくる、キャラクター性能・シナジー性能の“型”**」として整理したものです。現行付近では Set 17「Space Gods」があり、Set はおおむね数か月単位で入れ替わるため、細部の数値よりも**設計パターン**として見るのがよいです。TFT の Set はテーマ・ルール・チャンピオンがまとまった単位で、過去には Set 6 のオーグメント導入、Set 7 のドラゴン中心、Set 10 の音楽テーマ、Set 13 のアノマリー、Set 17 の神の供物システムなど、毎回別テーマで同型の遊びを作り直しています。 [\[https://jp...s/original\]](https://jp-prod.asyncgw.teams.microsoft.com/v1/objects/0-ejp-d3-19a0349f4718636a062ab355a44712d4/views/original), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Set), [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

## 1. 成長・捕食・合成型

### 代表例

* **Tahm Kench / Glutton**
* 別ユニットを食べて恒久ステータスを得るタイプ
* Cho’Gath 系の「倒す・食べる・大きくなる」系
* Set 17 の Dark Star のような「敵を消費する」演出・処刑型

この型は、**盤面外リソースやユニットそのものを燃料にして、1体をレイドボス化する**設計です。典型例は Set 6 の Tahm Kench で、Glutton により計画フェーズごとに味方を食べさせ、AD/AP/HP/防御系ステータスを永久に得られました。 [\[games.gg\]](https://games.gg/teamfight-tactics/guides/tft-set-17-space-gods-overview/), [\[lolchess.gg\]](https://lolchess.gg/champions/set6/tahmkench?hl=en)

現行寄りの Set 17 でも Tahm Kench は Brawler/Oracle のタンク寄りユニットとして登場し、直接の「味方を食べる」仕様ではなく、回復・シールド・報酬付与に寄った形になっています。つまり同じチャンピオンでも、**捕食成長型 → 報酬タンク型**のように役割が再解釈されます。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton), [\[mobalytics.gg\]](https://mobalytics.gg/tft/champions/tahmkench)

**強み**

* 完成時のロマンが大きい
* アイテム・星上げ・餌の質で上振れが作れる
* 1体キャリーが好きな人に向く

**弱み**

* 育成に時間・素材・HPを使う
* CCや処刑、バックライン火力に弱いことがある
* 途中で引けないと戦略が破綻しやすい

***

## 2. 防御し続ける・要塞化タンク型

### 代表例

* Galio
* Taric
* Shen
* Rammus
* The Mighty Mech
* Vanguard / Bastion / Brawler 系

この型は、**硬くなる、挑発する、シールドを張る、時間を稼ぐ**ことを主目的にした設計です。Set 11 の Galio は周囲の敵を挑発し、防御力を得て、最後に周囲へダメージと味方シールドを与えるタンクでした。 [\[tftactics.gg\]](https://tftactics.gg/champions/tahm_kench/), [\[metasrc.com\]](https://www.metasrc.com/tft/14.7/champion/galio)

Set 17 の The Mighty Mech もかなり近い型で、防御スタンスに入り、Durability を得て、近くの敵弾を引き寄せ、回復し、終了時に衝撃波を出します。これはまさに「ずっと防御し続けるキャラクター」の現代版に近いです。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-11-guide-how-to-play-storyweaver/)

また、Brawler はチーム全体にHPを与え、Brawler 自身はさらに多くの最大HPを得るという、TFTで非常に頻出する「前衛の厚みを増やす」系シナジーです。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 後衛キャリーを守りやすい
* 戦闘時間を伸ばしてスケーリング系と相性がよい
* 配置勝負が分かりやすい

**弱み**

* Giant Slayer 系、割合ダメージ、Shred/Sunder に弱い
* 火力が足りないと引き分け気味に負ける
* 無視される配置やバックラインアクセスに弱い

## 3. クエスト・達成報酬型

### 代表例

* Set 17 Aurelion Sol / God of Wonders
* 8-bit
* Piltover 系
* Anima 系研究
* Fortune / Mercenary / Underground の派生

この型は、**特定条件を満たすと報酬が出る**タイプです。Set 17 の Aurelion Sol は「50ゴールド到達」「6つの非重複トレイトを出す」「レベル9到達」などの試練を選び、達成すると報酬を得る“クエスト型”の神です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

Set 10 の 8-bit も、チームがダメージを出すほどスコアが伸び、ハイスコア更新で攻撃力が増え、最終的に大報酬を狙う型でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の Anima も、敗北やキルで Tech を貯め、100 Tech ごとに武器を研究・取得するため、これも「進捗バーを貯めるシナジー」と見なせます。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* ゲームプランが明確
* 達成時の報酬が大きい
* 条件達成のために構成を工夫する面白さがある

**弱み**

* 条件に縛られて盤面が弱くなりがち
* 達成前にHPが尽きるリスク
* 途中で方針転換しづらい

## 4. 連敗・賭け・キャッシュアウト型

### 代表例

* Fortune
* Mercenary
* Underground
* Piltover
* Heartsteel
* Set 17 Evelynn / God of Temptation

これはTFTを象徴する型の一つで、**序盤に負ける・HPを払う・不安定な盤面を許容する代わりに、後で大きな報酬を得る**設計です。Set 6 の Mercenary は宝箱にルートを貯め、勝利時に開封する仕組みでした。 [\[games.gg\]](https://games.gg/teamfight-tactics/guides/tft-set-17-space-gods-overview/)

Set 10 の Heartsteel は、敵を倒す・負けることで Hearts を稼ぎ、数ラウンドごとに報酬へ変換する経済シナジーで、Fortune / Underground / Piltover の精神的後継として説明されています。 [\[estnn.com\]](https://estnn.com/all-tft-set-10-traits-cheat-sheet/)

Set 17 の Evelynn / God of Temptation は、追加HP損失やショップ制限などの代償と引き換えに報酬を得るため、かなり直接的な「悪魔の取引」型です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

**強み**

* 上振れたときの爆発力が最大級
* 観戦・プレイ体験として派手
* 経済管理の腕が出る

**弱み**

* 安定性が低い
* 1回の誤勝利・誤敗北で崩れることがある
* 初心者にはHP管理が難しい

## 5. 召喚・追加ユニット型

### 代表例

* Cultist Galio
* Storyweaver Kayle
* Mecha / Mighty Mech
* Shepherd
* Country Dreadsteed
* Nomsy / Dragon 系

この型は、**盤面のユニット数や星数、シナジー段階に応じて追加戦力を呼ぶ**タイプです。Set 4 の Cultist は一定数を出すと Galio を召喚する代表例で、段階が上がるほど Galio が巨大化・強化されました。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-10-guide-how-to-play-heartsteel/)

Set 10 の Country は、味方チームが一定割合のHPを失うと Dreadsteed が突撃し、Country ユニットの星数で強くなる設計でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の Shepherd は、Bia/Bayin という助力存在を召喚し、Shepherd の合計星レベルで強化されるため、これも現行の召喚・ペット型に当たります。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 盤面が増えて前線・火力が補われる
* 星上げの価値が分かりやすい
* 高段階到達時の見た目が派手

**弱み**

* 召喚体依存で本体が弱い場合がある
* AoEにまとめて処理されることがある
* 高段階前提だと構成が縛られる

## 6. 変身・巨大化・特殊ユニット型

### 代表例

* Dragon 系
* Mecha 系
* Set 13 Viktor などの特殊コスト
* Set 17 The Mighty Mech
* Headliner / Chosen / Anomaly 強化ユニット

この型は、**通常ユニットとは違うルールで強い個体を作る**設計です。Set 7 Dragonlands はドラゴンを中心にした Set で、各種構成がドラゴンを主軸に組まれていました。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Set)

Set 13 では Anomaly により、4-6で1体を進化させ、特殊効果を付与する仕組みがありました。これは「1体をゲーム中盤で特別な存在にする」型です。 [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/)

Set 17 の Realm of the Gods も、神の供物を複数回選ぶことで後半に Boon を得るため、構成全体を特定方向へ変質させるメタ的な変身システムに近いです。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/), [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover)

**強み**

* 1体または1軸に明確なパワーを集中できる
* 構成の個性が出る
* 見た目・体験が分かりやすく楽しい

**弱み**

* その個体を引けないと弱い
* メタで対策されやすい
* 強すぎるとナーフされやすい

## 7. コピー・クローン・増殖型

### 代表例

* Meeple
* Woodland 系
* Trickster’s Glass 系
* Double Trouble 系
* Set 17 Replicator / Meeple

この型は、**同じユニット・追加スロット・コピー・分身を使って盤面価値を増やす**タイプです。Set 17 の Meeple は Meeps によって能力を強化し、さらに高段階ではベンチにクローンスロットを作り、配置したチャンピオンのコピーやゴールドを得る効果があります。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

この系統は「ショップ運を補う」「同一キャリーを複数使う」「星上げを早める」方向に働きやすく、TFTの“駒を集めるゲーム性”と相性がよいです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 盤面密度が上がる
* 星上げや複数キャリーに繋がる
* 上振れ時に一気に強くなる

**弱み**

* コピー先が弱いと効果も弱い
* ベンチ管理が難しい
* 終盤の単体性能に劣ることがある

## 8. 後衛砲台・スナイパー・遠距離キャリー型

### 代表例

* Sniper
* Longshot
* Ranger
* Cannoneer
* Big Shot
* Jhin / Xayah / Caitlyn 系

この型は、**前衛が時間を稼ぎ、後衛が遠距離から高火力を出す**TFTの基本形です。Set 17 でも Sniper や Fateweaver などの遠距離火力ユニットが存在し、Jhin、Xayah、Caitlyn などがこの文脈で使われます。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion), [\[mobalytics.gg\]](https://mobalytics.gg/tft/champions)

この型は毎 Set ほぼ必ず存在します。理由は単純で、TFTの構成理解が「前衛 + 後衛キャリー + 補助シナジー」という形に落とし込みやすいからです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion)

**強み**

* 構成が分かりやすい
* アイテム方針が明確
* 初心者から上級者まで使いやすい

**弱み**

* アサシン・ローグ・範囲CCに弱い
* 前衛が溶けると即崩壊
* 配置差で勝敗が大きく変わる

## 9. バックライン侵入・暗殺型

### 代表例

* Assassin
* Rogue
* Hacker
* Crowd Diver
* Shade
* Pyke / Akali / Zed / Katarina 系

この型は、**敵の後衛キャリーを直接狙う**タイプです。Set 17 でも Rogue が存在し、体力が一定以下になると影に入るような生存・奇襲系の性質を持っています。 [\[metatft.com\]](https://www.metatft.com/units/Galio), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion)

TFTでは後衛キャリー型が常に存在するため、それに対するメタとして暗殺型も繰り返し登場します。つまり「後衛砲台型」と「バックライン侵入型」は、かなり弁証法的な関係です。片方が流行ると、もう片方の価値が上がります。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 敵キャリーを先に倒せる
* 配置勝ちで格上に勝てる
* アイテムが噛み合うと爆発力が高い

**弱み**

* タンクが多い環境では刺さりにくい
* CCに弱い
* 配置読みを外すと弱い

Set 13 の Anomaly のように、特定キャリーを中盤で強化するシステムとも相性が良く、「この1体にマナ・AP・耐久を集めて回す」という形になりやすいです。 [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/)

**強み**

**弱み**

* マナ阻害やバーストに弱い

## 11. アイテム・装備変換型

### 代表例

* Scrap
* Cybernetic
* Artifact / Radiant 系
* Kayle / God of Order
* Blacksmith / Ornn 系

この型は、**アイテムの数・種類・強化によってパワーを得る**設計です。Set 17 の Kayle / God of Order は、ランダムな完成アイテムを Radiant にする、または構成に必要な素材を供給する方向の神です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

また、TFT全体としてチャンピオンはアイテムを装備して強化され、購入・売却・配置・星上げと組み合わさるため、アイテム変換型はゲームの根幹に近いシナジーです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/)

**強み**

* アイテム事故を緩和できる
* 高品質アイテムで終盤が強い
* 構成の柔軟性が高い

**弱み**

* アイテムが噛み合わないと弱い

## 12. 配置・地形・範囲指定型

### 代表例

* Disco Ball
* Elemental Hex
* Socialite Hex
* N.O.V.A. selector
* Yasuo / God of Abyss 系のヘックス強化
* Galio / Rammus 系の範囲挑発

この型は、**どこに置くかで性能が大きく変わる**タイプです。Set 10 の Disco はディスコボール周辺の味方に攻撃速度や回復を与えるため、配置そのものがシナジーの一部でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の N.O.V.A. は特定チャンピオンの Strike を選択してパワーサージ時に発動させるため、構成と配置・選択が絡むシナジーです。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 配置の上手さで差が出る
* 同じ構成でもプレイヤースキルが出る
* 対戦相手ごとに調整できる

**弱み**

* スカウトが必要
* Zephyr/Shroud 的な妨害に弱い
* 初心者には負担が大きい

## 13. 時間経過・スケーリング型

* Ascension 系

防御型タンクや回復型サポートと相性がよく、逆にバースト構成には弱いです。The Mighty Mech のような防御スタンス型や、Tahm Kench のような回復タンクは、このスケーリング型を支える前衛として機能します。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-11-guide-how-to-play-storyweaver/), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton)

**強み**

* 長期戦で非常に強い
* タンク・回復・シールドと相性がよい
* 終盤盤面で安定しやすい

**弱み**

* 初動バーストに弱い
* キャリーが早く落ちると何もできない
* 戦闘時間を稼げる前衛が必須

## 14. 状態異常・妨害・制御型

### 代表例

* Stun
* Taunt
* Charm
* Knock-up
* Shred / Sunder
* Disruptor
* Galio / Sejuani / Nautilus / Morgana 系

この型は、**敵に行動させない、火力を下げる、防御を削る**ことを目的にします。Galio の挑発・防御上昇・範囲ダメージ・味方シールドは、タンク型でありながら妨害型でもあります。 [\[tftactics.gg\]](https://tftactics.gg/champions/tahm_kench/), [\[op.gg\]](https://op.gg/tft)

Set 17 でも N.O.V.A. の Aatrox が敵に Shred/Sunder を与えるように、防御低下をチーム全体の火力補助として使う設計が見られます。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 格上相手にも勝ち筋を作れる
* 敵キャリーの初回スキルを遅らせられる
* 味方火力を間接的に伸ばせる

**弱み**

* CC無効・Unstoppable に弱い
* ダメージそのものは不足しがち
* 対象や範囲がズレると弱い

| 変身型 | 特殊個体・進化体を作る      | Dragon、Mecha、Anomaly、Chosen/Headliner             |
| 砲台型 | 後衛が安全に火力を出す      | Sniper、Longshot、Jhin、Xayah                        |
| 奇襲型 | 後衛を直接狙う          | Assassin、Rogue、Zed、Akali                          |
| 経済型 | ゴールド・XP・アイテムを増やす | Ahri神、Mercenary、Heartsteel、Oracle Tahm Kench      |

***

## 私見としての整理

TFTのSet設計は、完全に新しいものを毎回作っているというより、**既存の遊びの型を、テーマに合わせて言い換えている**ことが多いです。

たとえば、

* 「別キャラを食べる」  
  → **育成型 + リソース変換型**
* 「ずっと防御する」  
  → **要塞型 + 時間稼ぎ型**
* 「クエストを攻略する」  
  → **達成報酬型 + 経済/アイテム型**
* 「負けて報酬を貯める」  
  → **賭け型 + キャッシュアウト型**
* 「神を選んで供物を得る」  
  → **メタ進行型 + クエスト/経済型**

という感じです。

なので新Setを見るときは、チャンピオン名やシナジー名よりも、

1. **これは経済を増やす型か**
2. **戦闘中に強くなる型か**
3. **盤面外で進捗を貯める型か**
4. **1体を化け物にする型か**
5. **召喚体やコピーで盤面を増やす型か**
6. **配置で性能が変わる型か**
7. **負けを報酬に変える型か**

で見ると、かなり早く理解できます。










以下では、TFTやDota Auto Chess、Auto Chess、Battlegrounds系などの**オートチェス系ゲームにおけるキャラクター／ユニット性能・メカニズムの類型**を、できるだけ網羅的に整理します。  
セットやタイトルごとに名称は違いますが、だいたい同じ設計要素に分解できます。

***

# 1. 基本性能による類型

## 1.1 コスト・レアリティ型

ユニット性能はまずコスト帯で大きく分類されます。

| 類型          | 役割          | 特徴                   |
| ----------- | ----------- | -------------------- |
| 低コスト序盤型     | 序盤の盤面形成     | 早く重ねやすいが終盤は失速しやすい    |
| 低コストリロール型   | 星3で主力化      | 星3前提で高火力・高耐久になる      |
| 中コスト中核型     | 中盤の安定化      | 構成の橋渡し役、サブキャリーになりやすい |
| 高コスト終盤型     | レイトゲームの主力   | 単体性能・スキル性能が高い        |
| レジェンダリー／特殊枠 | 構成を壊す・完成させる | 強力だが入手難度が高い          |

### 見方

* **低コストが強い設計**  
  → リロール戦略が成立しやすい。
* **高コストが強い設計**  
  → レベル先行・経済管理が重要になる。
* **中コストが強い設計**  
  → 7〜8レベル周辺でゲームが決まりやすい。

***

## 1.2 ステータス分布型

同じコストでも、ステータスの偏りによって役割が変わります。

| 類型       | 特徴               |
| -------- | ---------------- |
| 高HP型     | 前衛向き。スキル発動まで耐える  |
| 高防御・高魔防型 | 物理／魔法に強いタンク      |
| 高攻撃力型    | 通常攻撃キャリー向き       |
| 高攻撃速度型   | オンヒット、マナ獲得、DPS向き |
| 高射程型     | 後衛キャリー向き         |
| 低射程高火力型  | アサシン・近接キャリー向き    |
| 高マナコスト型  | 一発が重いが発動が遅い      |
| 低マナコスト型  | スキル回転率が高い        |

***

# 2. 戦闘ロールによる類型

## 2.1 タンク系

### メインタンク

前衛で最も攻撃を受けるユニット。

**特徴**

* 高HP
* 防御・魔防が高い
* シールド、回復、ダメージ軽減を持つ
* CCを受けても役割を果たしやすい

**強み**

* 後衛キャリーの時間を稼ぐ
* 構成の安定性が高い

**弱み**

* 火力は低め
* 防御無視、割合ダメージ、重傷に弱い

***

### サブタンク

メインタンクほど硬くないが、前線維持に貢献するユニット。

**例**

* 防御系シナジーを持つ中コスト
* CC付き前衛
* 回復・シールドを持つファイター

***

### オフタンク／ファイター

耐久と火力を両立する前衛。

**特徴**

* 攻撃しながら回復する
* 殴り合いに強い
* アイテム次第でキャリー化する

**強み**

* タンクと火力を兼ねる
* 少人数戦に強い

**弱み**

* 瞬間火力に落とされると何もできない
* 重傷やCCに弱い

***

## 2.2 キャリー系

### AD通常攻撃キャリー

通常攻撃を主なダメージ源とするユニット。

**特徴**

* 攻撃速度・攻撃力・クリティカルが重要
* 長期戦でDPSが伸びる
* 後衛配置が多い

**強み**

* 継続火力が高い
* タンク処理が安定しやすい

**弱み**

* アサシン、バックライン攻撃、CCに弱い
* 攻撃対象を誤ると火力が無駄になる

***

### APスキルキャリー

スキルによる魔法ダメージが主力。

**特徴**

* マナアイテム、魔力アイテムと相性が良い
* 範囲攻撃や高倍率スキルが多い
* スキル発動タイミングが重要

**強み**

* 一撃で盤面を破壊できる
* 後衛一掃や範囲処理に強い

**弱み**

* 発動前に倒されると弱い
* 魔防、シールド、マナ阻害に弱い

***

### バーストキャリー

短時間で敵を倒すことに特化したキャリー。

**特徴**

* 高倍率スキル
* クリティカル
* 初回スキルが強い
* 後衛狙いが多い

**強み**

* 敵キャリーを即座に処理できる
* 短期決戦に強い

**弱み**

* 倒しきれないと失速する
* タンク構成に弱い場合がある

***

### スケーリングキャリー

時間経過や攻撃回数で強くなるキャリー。

**例**

* 攻撃するたび攻撃速度上昇
* スキル使用ごとに強化
* キルするたび強化
* 戦闘時間が長いほどダメージ増加

**強み**

* 長期戦で圧倒的
* タンク構成と相性が良い

**弱み**

* 序盤の爆発力が低い
* バーストやCCに弱い

***

### ハイパーキャリー

アイテム・星・シナジーを集中させて勝ち筋にするユニット。

**特徴**

* 構成全体がそのユニットを守る設計
* 補助、前衛、CCがキャリーのために存在する
* 完成時の勝率が高い

**弱み**

* 依存度が高い
* 引けない、重ならない、アイテムが合わないと崩れる

***

## 2.3 サポート系

### バッファー

味方を強化するユニット。

**効果例**

* 攻撃速度上昇
* 攻撃力上昇
* 魔力上昇
* 防御・魔防上昇
* クリティカル率上昇
* 与ダメージ増加

***

### ヒーラー

味方を回復するユニット。

**強み**

* 継戦能力が高い
* タンク構成と相性が良い

**弱み**

* 重傷、回復阻害に弱い
* バーストで即死すると意味が薄い

***

### シールダー

味方にシールドを付与するユニット。

**特徴**

* 回復阻害の影響を受けにくいことが多い
* 瞬間耐久を上げる
* 範囲シールドなら盤面全体を守れる

***

### デバッファー

敵を弱体化させるユニット。

**効果例**

* 防御低下
* 魔防低下
* 攻撃速度低下
* 与ダメージ低下
* 被ダメージ増加
* 回復阻害
* マナ阻害

***

# 3. スキルメカニズムによる類型

## 3.1 ダメージスキル

### 単体高火力

1体に大ダメージ。

**得意**

* タンク処理
* 敵キャリー暗殺

**苦手**

* 召喚物や低価値ユニットに吸われること

***

### 範囲ダメージ

複数体にダメージ。

| 種類       | 特徴                |
| -------- | ----------------- |
| 円形範囲     | 密集に強い             |
| 直線範囲     | 配置読みが重要           |
| 扇形範囲     | 前衛から後衛に届きやすい      |
| 全体攻撃     | 安定して全員に影響         |
| ランダム複数対象 | 配置に左右されにくいが狙いが不安定 |

***

### 継続ダメージ

時間をかけて削るタイプ。

**例**

* 毒
* 炎上
* 出血
* 感電
* DoTフィールド

**強み**

* 高HP相手に強い
* 長期戦向き

**弱み**

* 即効性が低い
* 回復やシールドに相殺されやすい

***

### 割合ダメージ

最大HPまたは現在HPに応じてダメージ。

**強み**

* タンクに強い
* 防御寄り環境で価値が高い

**弱み**

* 低HPキャリー処理では過剰性能になりにくい
* 防御無視かどうかで評価が大きく変わる

***

### 確定ダメージ／防御無視

防御や魔防を無視するダメージ。

**強み**

* どんな相手にも通りやすい
* タンク環境で強い

**弱み**

* 数値が低めに設定されやすい
* バランス調整で弱体化されやすい

***

## 3.2 CC、行動阻害型

### スタン

一定時間行動不能。

### ノックアップ

行動不能＋位置や対象を崩す場合がある。

### サイレンス

スキル発動を封じる。

### 武装解除

通常攻撃を封じる。

### 凍結・拘束

移動や攻撃を止める。

### 挑発

敵の攻撃対象を強制する。

### 恐怖・逃走

敵を遠ざけたり行動不能にする。

### マナリーブ／マナバーン

敵のマナ獲得やスキル発動を妨害する。

**CC型の強み**

* 格上相手にも勝ち筋を作れる
* 敵キャリーの初回スキルを遅らせられる

**弱み**

* CC無効、クイックシルバー系、状態異常耐性に弱い
* ダメージ不足になりやすい

***

## 3.3 防御・生存スキル

### シールド

一定ダメージを吸収。

### ダメージ軽減

受けるダメージを割合で減らす。

### 無敵

短時間ダメージを受けない。

### 回復

HPを回復する。

### ライフスティール／オムニヴァンプ

与えたダメージに応じて回復。

### 復活

倒された後に再度戦闘へ復帰。

### 分身・身代わり

攻撃対象をずらす。

### ターゲット不可

一時的に狙われない。

***

## 3.4 移動・位置操作型

### ダッシュ

敵に接近する、または安全位置へ移動する。

### ブリンク／瞬間移動

後衛に飛び込むことが多い。

### 引き寄せ

敵を自陣側へ引っ張る。

### 押し出し

敵前衛をどかす、敵を散らす。

### 位置交換

敵味方の位置を入れ替える。

### 後衛狙い

通常のターゲットルールを無視して後ろを狙う。

**位置操作型の重要性**

* オートチェスでは操作ができないため、位置操作は非常に強いメカニズムです。
* ただし、配置読みや対策配置で価値が大きく変わります。

***

# 4. ターゲット選択による類型

ユニットが「誰を攻撃するか」は性能を大きく左右します。

| ターゲット型     | 特徴             |
| ---------- | -------------- |
| 最も近い敵      | 基本型。前衛に吸われやすい  |
| 現在攻撃中の敵    | 安定するが柔軟性は低い    |
| 最もHPが低い敵   | キル回収に強い        |
| 最もHPが高い敵   | タンク処理向き        |
| 最も遠い敵      | 後衛狙いになりやすい     |
| 攻撃力が最も高い敵  | 物理キャリー対策       |
| 魔力が最も高い敵   | APキャリー対策       |
| ランダム対象     | 安定性は低いが予測されにくい |
| 直線上の敵      | 配置依存が高い        |
| 周囲の敵       | 近接範囲型          |
| 最多人数に当たる位置 | 高性能AI型スキルに多い   |

***

# 5. ダメージタイプによる類型

## 5.1 物理ダメージ型

攻撃力、防御低下、クリティカルと相性が良い。

**代表的な役割**

* 通常攻撃キャリー
* 近接ファイター
* スナイパー
* アサシン

***

## 5.2 魔法ダメージ型

魔力、マナ、魔防低下と相性が良い。

**代表的な役割**

* メイジ
* ソーサラー
* 範囲スキルキャリー

***

## 5.3 確定ダメージ型

防御・魔防を無視する。

**代表的な役割**

* 高コスト特殊キャリー
* タンクキラー
* 条件達成型ユニット

***

## 5.4 混合ダメージ型

物理と魔法、または確定ダメージを併せ持つ。

**強み**

* 対策されにくい

**弱み**

* アイテム最適化が難しい場合がある

***

# 6. スケーリング方式による類型

## 6.1 星依存型

星2、星3になることで大きく性能が伸びる。

**向いている戦略**

* リロール構成
* 低コスト主力
* 同一ユニットを大量に集める戦略

***

## 6.2 レベル依存型

高レベルで高コストを入れるほど強くなる。

**向いている戦略**

* ファスト8
* ファスト9
* 高コスト詰め合わせ構成

***

## 6.3 アイテム依存型

特定アイテムがないと真価を発揮しにくい。

**例**

* 攻撃速度アイテム必須
* マナアイテム必須
* 防御アイテム必須
* クリティカルアイテム必須

***

## 6.4 シナジー依存型

特定の特性や種族、職業の発動で強くなる。

**例**

* 6体以上で大幅強化
* 縦に深く伸ばすと強い
* 少数発動でも優秀
* エンブレムで壊れる

***

## 6.5 時間依存型

戦闘が長引くほど強くなる。

**例**

* 攻撃速度が増える
* スタックが増える
* 与ダメージが増える
* 回復量が増える

***

## 6.6 キル依存型

敵を倒すことで強くなる。

**例**

* キルでマナ回復
* キルで再発動
* キルでステータス上昇
* キルで追加行動

**特徴**

* 勝ち始めると連鎖的に強い
* 最初の1体を倒せないと弱い

***

# 7. シナジー・特性における類型

## 7.1 縦伸ばし型

同じ特性を多く集めるほど強くなる。

**例**

* 2 / 4 / 6 / 8 のように段階強化
* 最大段階で爆発的に強い

**強み**

* 完成時の出力が高い

**弱み**

* 柔軟性が低い
* 必要ユニットを引けないと弱い

***

## 7.2 横広げ型

複数の小シナジーを組み合わせる。

**強み**

* 柔軟
* アイテムや引きに対応しやすい
* 中盤が安定しやすい

**弱み**

* 最大火力が縦構成に劣ることがある

***

## 7.3 キャリー専用強化型

特定ロールを大きく強化する。

**例**

* 射手強化
* 魔法使い強化
* アサシン強化
* ファイター強化

***

## 7.4 全体バフ型

盤面全体に効果がある。

**例**

* 全員のHP増加
* 全員の攻撃速度増加
* 全員の防御上昇

***

## 7.5 条件付きシナジー型

発動条件や配置条件がある。

**例**

* 隣接している味方を強化
* 同じ列の味方を強化
* 特定位置のユニットを強化
* 戦闘開始時のみ強化
* 一定時間後に強化

***

## 7.6 召喚・追加ユニット型

戦闘中に追加戦力を出す。

**強み**

* 数的有利を作れる
* 敵のターゲットを分散できる

**弱み**

* 範囲攻撃に弱い
* 召喚物が弱いと価値が低い

***

# 8. 攻撃射程・位置取りによる類型

## 8.1 近接前衛

前に出て戦う。

**役割**

* 近接キャリー

## 8.2 中距離ユニット

## 8.3 遠距離後衛

***

## 8.4 潜入・暗殺型

後衛にアクセスする。

**弱み**

* 配置対策されやすい

# 9. テンポ・ゲーム段階による類型

## 9.1 序盤番長型

## 9.4 連勝向きユニット

## 9.5 連敗・逆転向きユニット

後で完成すると強い。

**特徴**

## 10.2 APアイテム適性型

**欲しい要素**

* スキルクリティカル
* 魔防低下

**該当ユニット**

***

## 10.3 タンクアイテム適性型

**欲しい要素**

* HP
* 防御
* 魔防
* ダメージ軽減
* シールド
* 回復増加

**該当ユニット**

* メインタンク
* 挑発持ち
* 長時間生存で価値が出るユニット

***

## 10.4 マナアイテム適性型

**特徴**

* 初回スキルが重要
* スキル回転率で強くなる
* CCや範囲火力と相性が良い

***

## 10.5 特定アイテム必須型

特定アイテムがないと性能が大きく落ちる。

**例**

* 攻撃速度がないと弱い
* 回復手段がないと前衛で生き残れない
* マナ加速がないと初回スキルが遅すぎる
* 防御貫通がないとタンクを抜けない

***

# 11. リスク・安定性による類型

## 11.1 安定型

毎試合ある程度の仕事をする。

**例**

* 広範囲CC
* 全体バフ
* タンク
* 後衛通常攻撃キャリー

***

## 11.2 ハイロール型

条件が揃うと非常に強い。

**例**

* 星3高コスト
* 特殊アイテム依存
* レアシナジー最大発動
* キル連鎖型キャリー

***

## 11.3 配置依存型

配置が合えば強いが、外すと弱い。

**例**

* 直線スキル
* 後衛飛び込み
* 隣接バフ
* フック系
* 前方扇形スキル

***

## 11.4 相性依存型

相手構成によって強さが変わる。

**例**

* 範囲火力：密集相手に強い
* 単体火力：少数精鋭に強い
* 魔法防御低下：AP構成で強い
* 重傷：回復構成に強い
* アサシン：後衛キャリー構成に強い

***

# 12. 特殊メカニズムによる類型

## 12.1 変身型

一定条件で姿や性能が変わる。

**例**

* HP低下で変身
* マナ最大で変身
* 一定時間後に変身
* 星やシナジーで形態変化

***

## 12.2 コピー・複製型

敵や味方をコピーする。

**例**

* 味方の能力を複製
* 敵のスキルをコピー
* 分身を生成
* 同名ユニットを追加召喚

***

## 12.3 盗む・奪う型

敵の能力やリソースを奪う。

**例**

* 攻撃力を奪う
* 防御を奪う
* マナを奪う
* HPを吸収する

***

## 12.4 スタック蓄積型

戦闘中または試合中にスタックを貯める。

**例**

* 永続HP増加
* 永続攻撃力増加
* ラウンドごとの強化
* キル数に応じた強化

***

## 12.5 経済干渉型

戦闘力以外で有利を作る。

**例**

* ゴールド生成
* アイテム生成
* ショップ操作
* 経験値補助
* 戦利品追加
* 勝敗時報酬変化

これはキャラクターというより、特性やオーグメントに多いですが、ユニット能力として存在することもあります。

***

## 12.6 ルール破壊型

通常ルールを変えるタイプ。

**例**

* 配置数上限を増やす
* 特性カウントを追加する
* アイテム枠を増やす
* 死亡後も効果を残す
* 戦闘開始前に敵を弱体化する
* 同じユニットを複数扱いにする

***

# 13. メタ上の役割による類型

## 13.1 キャリー

勝敗を決める主力。

## 13.2 アイテムホルダー

本命キャリーが来るまでアイテムを持つ仮の主力。

## 13.3 シナジーボット

## 13.4 CC要員

火力より妨害目的で採用。

## 13.5 タンク要員

前線維持のために採用。

## 13.6 テックユニット

特定相手への対策として入れる。

**例**

* 回復対策
* アサシン対策
* 魔法対策
* 物理対策
* 密集対策
* 後衛対策

## 13.7 移行用ユニット

序盤から終盤構成へつなぐための中継役。

# 14. 構成単位で見たキャラクター類型

## 14.1 フロント・トゥ・バック型

**必要要素**

* 強い前衛
* 後衛DPS
* 防御貫通

## 14.2 バックラインアクセス型

敵後衛を直接倒す構成。

**必要要素**

## 14.3 AoE一掃型

**必要要素**

* 範囲魔法
* CC
* マナ加速
* 敵を固める手段

***

## 14.4 耐久・回復型

倒されずに勝つ構成。

**必要要素**

* タンク
* 回復
* シールド
* ダメージ軽減

## 14.5 召喚・物量型

* 範囲バフ

## 14.6 ワンショット型

初回スキルや開幕効果で一気に倒す構成。

**必要要素**

* マナ加速

# 15. 評価するときのチェックリスト

あるユニットを評価するときは、次の観点で見ると整理しやすいです。

## 基本性能

* コストに対してステータスは高いか
* 星2、星3でどれくらい伸びるか
* 射程は長いか
* マナは重いか軽いか

## スキル性能

* 単体か範囲か
* 初回発動が早いか
* 対象指定は賢いか
* ダメージ以外の効果があるか
* CCや防御効果を持つか

## アイテム適性

* AD、AP、タンク、マナのどれが合うか
* 必須アイテムがあるか
* 代替アイテムでも機能するか

## 構成適性

* 縦シナジー向きか
* 横シナジー向きか
* メインキャリーかサブキャリーか
* シナジーボットでも価値があるか

## メタ適性

* タンク環境に強いか
* バースト環境に強いか
* 回復環境に強いか
* アサシン環境に弱いか
* CC環境に弱いか

## 実戦的に一番重要な見方

個人的には、ユニットを見るときは次の5軸で考えるのが一番実用的です。

1. **誰がダメージを出すか**
2. **誰が時間を稼ぐか**
3. **誰が敵の行動を止めるか**
4. **どのアイテムを誰に持たせるか**
5. **そのユニットは序盤・中盤・終盤のどこで強いか**

この5つを押さえると、単なる「強い／弱い」ではなく、  
「このユニットは強いが、前衛が足りないと機能しない」  
「このユニットは火力は低いが、CC要員として採用価値がある」  
「このキャリーはアイテムが揃わないなら別の構成に移行した方がいい」  
というふうに、かなり正確に判断できます。
