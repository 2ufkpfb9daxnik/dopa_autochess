以下は「**TFTで何度も形を変えて出てくる、キャラクター性能・シナジー性能の“型”**」として整理したものです。現行付近では Set 17「Space Gods」があり、Set はおおむね数か月単位で入れ替わるため、細部の数値よりも**設計パターン**として見るのがよいです。TFT の Set はテーマ・ルール・チャンピオンがまとまった単位で、過去には Set 6 のオーグメント導入、Set 7 のドラゴン中心、Set 10 の音楽テーマ、Set 13 のアノマリー、Set 17 の神の供物システムなど、毎回別テーマで同型の遊びを作り直しています。 [\[https://jp...s/original\]](https://jp-prod.asyncgw.teams.microsoft.com/v1/objects/0-ejp-d3-19a0349f4718636a062ab355a44712d4/views/original), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Set), [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

## 1. 成長・捕食・合成型

### 代表例

* **Tahm Kench / Glutton**
* 別ユニットを食べて恒久ステータスを得るタイプ
* Cho’Gath 系の「倒す・食べる・大きくなる」系
* Set 17 の Dark Star のような「敵を消費する」演出・処刑型

この型は、**盤面外リソースやユニットそのものを燃料にして、1体をレイドボス化する**設計です。典型例は Set 6 の Tahm Kench で、Glutton により計画フェーズごとに味方を食べさせ、AD/AP/HP/防御系ステータスを永久に得られました。 [\[games.gg\]](https://games.gg/teamfight-tactics/guides/tft-set-17-space-gods-overview/), [\[lolchess.gg\]](https://lolchess.gg/champions/set6/tahmkench?hl=en)

現行寄りの Set 17 でも Tahm Kench は Brawler/Oracle のタンク寄りユニットとして登場し、直接の「味方を食べる」仕様ではなく、回復・シールド・報酬付与に寄った形になっています。つまり同じチャンピオンでも、**捕食成長型 → 報酬タンク型**のように役割が再解釈されます。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton), [\[mobalytics.gg\]](https://mobalytics.gg/tft/champions/tahmkench)

**強み**

* 完成時のロマンが大きい
* アイテム・星上げ・餌の質で上振れが作れる
* 1体キャリーが好きな人に向く

**弱み**

* 育成に時間・素材・HPを使う
* CCや処刑、バックライン火力に弱いことがある
* 途中で引けないと戦略が破綻しやすい

***

## 2. 防御し続ける・要塞化タンク型

### 代表例

* Galio
* Taric
* Shen
* Rammus
* The Mighty Mech
* Vanguard / Bastion / Brawler 系

この型は、**硬くなる、挑発する、シールドを張る、時間を稼ぐ**ことを主目的にした設計です。Set 11 の Galio は周囲の敵を挑発し、防御力を得て、最後に周囲へダメージと味方シールドを与えるタンクでした。 [\[tftactics.gg\]](https://tftactics.gg/champions/tahm_kench/), [\[metasrc.com\]](https://www.metasrc.com/tft/14.7/champion/galio)

Set 17 の The Mighty Mech もかなり近い型で、防御スタンスに入り、Durability を得て、近くの敵弾を引き寄せ、回復し、終了時に衝撃波を出します。これはまさに「ずっと防御し続けるキャラクター」の現代版に近いです。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-11-guide-how-to-play-storyweaver/)

また、Brawler はチーム全体にHPを与え、Brawler 自身はさらに多くの最大HPを得るという、TFTで非常に頻出する「前衛の厚みを増やす」系シナジーです。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 後衛キャリーを守りやすい
* 戦闘時間を伸ばしてスケーリング系と相性がよい
* 配置勝負が分かりやすい

**弱み**

* Giant Slayer 系、割合ダメージ、Shred/Sunder に弱い
* 火力が足りないと引き分け気味に負ける
* 無視される配置やバックラインアクセスに弱い

## 3. クエスト・達成報酬型

### 代表例

* Set 17 Aurelion Sol / God of Wonders
* 8-bit
* Piltover 系
* Anima 系研究
* Fortune / Mercenary / Underground の派生

この型は、**特定条件を満たすと報酬が出る**タイプです。Set 17 の Aurelion Sol は「50ゴールド到達」「6つの非重複トレイトを出す」「レベル9到達」などの試練を選び、達成すると報酬を得る“クエスト型”の神です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

Set 10 の 8-bit も、チームがダメージを出すほどスコアが伸び、ハイスコア更新で攻撃力が増え、最終的に大報酬を狙う型でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の Anima も、敗北やキルで Tech を貯め、100 Tech ごとに武器を研究・取得するため、これも「進捗バーを貯めるシナジー」と見なせます。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* ゲームプランが明確
* 達成時の報酬が大きい
* 条件達成のために構成を工夫する面白さがある

**弱み**

* 条件に縛られて盤面が弱くなりがち
* 達成前にHPが尽きるリスク
* 途中で方針転換しづらい

## 4. 連敗・賭け・キャッシュアウト型

### 代表例

* Fortune
* Mercenary
* Underground
* Piltover
* Heartsteel
* Set 17 Evelynn / God of Temptation

これはTFTを象徴する型の一つで、**序盤に負ける・HPを払う・不安定な盤面を許容する代わりに、後で大きな報酬を得る**設計です。Set 6 の Mercenary は宝箱にルートを貯め、勝利時に開封する仕組みでした。 [\[games.gg\]](https://games.gg/teamfight-tactics/guides/tft-set-17-space-gods-overview/)

Set 10 の Heartsteel は、敵を倒す・負けることで Hearts を稼ぎ、数ラウンドごとに報酬へ変換する経済シナジーで、Fortune / Underground / Piltover の精神的後継として説明されています。 [\[estnn.com\]](https://estnn.com/all-tft-set-10-traits-cheat-sheet/)

Set 17 の Evelynn / God of Temptation は、追加HP損失やショップ制限などの代償と引き換えに報酬を得るため、かなり直接的な「悪魔の取引」型です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

**強み**

* 上振れたときの爆発力が最大級
* 観戦・プレイ体験として派手
* 経済管理の腕が出る

**弱み**

* 安定性が低い
* 1回の誤勝利・誤敗北で崩れることがある
* 初心者にはHP管理が難しい

## 5. 召喚・追加ユニット型

### 代表例

* Cultist Galio
* Storyweaver Kayle
* Mecha / Mighty Mech
* Shepherd
* Country Dreadsteed
* Nomsy / Dragon 系

この型は、**盤面のユニット数や星数、シナジー段階に応じて追加戦力を呼ぶ**タイプです。Set 4 の Cultist は一定数を出すと Galio を召喚する代表例で、段階が上がるほど Galio が巨大化・強化されました。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-10-guide-how-to-play-heartsteel/)

Set 10 の Country は、味方チームが一定割合のHPを失うと Dreadsteed が突撃し、Country ユニットの星数で強くなる設計でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の Shepherd は、Bia/Bayin という助力存在を召喚し、Shepherd の合計星レベルで強化されるため、これも現行の召喚・ペット型に当たります。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 盤面が増えて前線・火力が補われる
* 星上げの価値が分かりやすい
* 高段階到達時の見た目が派手

**弱み**

* 召喚体依存で本体が弱い場合がある
* AoEにまとめて処理されることがある
* 高段階前提だと構成が縛られる

## 6. 変身・巨大化・特殊ユニット型

### 代表例

* Dragon 系
* Mecha 系
* Set 13 Viktor などの特殊コスト
* Set 17 The Mighty Mech
* Headliner / Chosen / Anomaly 強化ユニット

この型は、**通常ユニットとは違うルールで強い個体を作る**設計です。Set 7 Dragonlands はドラゴンを中心にした Set で、各種構成がドラゴンを主軸に組まれていました。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Set)

Set 13 では Anomaly により、4-6で1体を進化させ、特殊効果を付与する仕組みがありました。これは「1体をゲーム中盤で特別な存在にする」型です。 [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/)

Set 17 の Realm of the Gods も、神の供物を複数回選ぶことで後半に Boon を得るため、構成全体を特定方向へ変質させるメタ的な変身システムに近いです。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/), [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover)

**強み**

* 1体または1軸に明確なパワーを集中できる
* 構成の個性が出る
* 見た目・体験が分かりやすく楽しい

**弱み**

* その個体を引けないと弱い
* メタで対策されやすい
* 強すぎるとナーフされやすい

## 7. コピー・クローン・増殖型

### 代表例

* Meeple
* Woodland 系
* Trickster’s Glass 系
* Double Trouble 系
* Set 17 Replicator / Meeple

この型は、**同じユニット・追加スロット・コピー・分身を使って盤面価値を増やす**タイプです。Set 17 の Meeple は Meeps によって能力を強化し、さらに高段階ではベンチにクローンスロットを作り、配置したチャンピオンのコピーやゴールドを得る効果があります。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

この系統は「ショップ運を補う」「同一キャリーを複数使う」「星上げを早める」方向に働きやすく、TFTの“駒を集めるゲーム性”と相性がよいです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 盤面密度が上がる
* 星上げや複数キャリーに繋がる
* 上振れ時に一気に強くなる

**弱み**

* コピー先が弱いと効果も弱い
* ベンチ管理が難しい
* 終盤の単体性能に劣ることがある

## 8. 後衛砲台・スナイパー・遠距離キャリー型

### 代表例

* Sniper
* Longshot
* Ranger
* Cannoneer
* Big Shot
* Jhin / Xayah / Caitlyn 系

この型は、**前衛が時間を稼ぎ、後衛が遠距離から高火力を出す**TFTの基本形です。Set 17 でも Sniper や Fateweaver などの遠距離火力ユニットが存在し、Jhin、Xayah、Caitlyn などがこの文脈で使われます。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion), [\[mobalytics.gg\]](https://mobalytics.gg/tft/champions)

この型は毎 Set ほぼ必ず存在します。理由は単純で、TFTの構成理解が「前衛 + 後衛キャリー + 補助シナジー」という形に落とし込みやすいからです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion)

**強み**

* 構成が分かりやすい
* アイテム方針が明確
* 初心者から上級者まで使いやすい

**弱み**

* アサシン・ローグ・範囲CCに弱い
* 前衛が溶けると即崩壊
* 配置差で勝敗が大きく変わる

## 9. バックライン侵入・暗殺型

### 代表例

* Assassin
* Rogue
* Hacker
* Crowd Diver
* Shade
* Pyke / Akali / Zed / Katarina 系

この型は、**敵の後衛キャリーを直接狙う**タイプです。Set 17 でも Rogue が存在し、体力が一定以下になると影に入るような生存・奇襲系の性質を持っています。 [\[metatft.com\]](https://www.metatft.com/units/Galio), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion)

TFTでは後衛キャリー型が常に存在するため、それに対するメタとして暗殺型も繰り返し登場します。つまり「後衛砲台型」と「バックライン侵入型」は、かなり弁証法的な関係です。片方が流行ると、もう片方の価値が上がります。 [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Champion), [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 敵キャリーを先に倒せる
* 配置勝ちで格上に勝てる
* アイテムが噛み合うと爆発力が高い

**弱み**

* タンクが多い環境では刺さりにくい
* CCに弱い
* 配置読みを外すと弱い

Set 13 の Anomaly のように、特定キャリーを中盤で強化するシステムとも相性が良く、「この1体にマナ・AP・耐久を集めて回す」という形になりやすいです。 [\[thegamehaus.com\]](https://thegamehaus.com/teamfight-tactics/every-tft-set-list-release-date-and-theme/2023/07/19/)

**強み**

**弱み**

* マナ阻害やバーストに弱い

## 11. アイテム・装備変換型

### 代表例

* Scrap
* Cybernetic
* Artifact / Radiant 系
* Kayle / God of Order
* Blacksmith / Ornn 系

この型は、**アイテムの数・種類・強化によってパワーを得る**設計です。Set 17 の Kayle / God of Order は、ランダムな完成アイテムを Radiant にする、または構成に必要な素材を供給する方向の神です。 [\[tftstratify.com\]](https://tftstratify.com/en_US/trait/TFT16_Piltover), [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-13-reveal-into-the-arcane/)

また、TFT全体としてチャンピオンはアイテムを装備して強化され、購入・売却・配置・星上げと組み合わさるため、アイテム変換型はゲームの根幹に近いシナジーです。 [\[progameguides.com\]](https://progameguides.com/teamfight-tactics/tft-fates-set-4-champions-and-traits/)

**強み**

* アイテム事故を緩和できる
* 高品質アイテムで終盤が強い
* 構成の柔軟性が高い

**弱み**

* アイテムが噛み合わないと弱い

## 12. 配置・地形・範囲指定型

### 代表例

* Disco Ball
* Elemental Hex
* Socialite Hex
* N.O.V.A. selector
* Yasuo / God of Abyss 系のヘックス強化
* Galio / Rammus 系の範囲挑発

この型は、**どこに置くかで性能が大きく変わる**タイプです。Set 10 の Disco はディスコボール周辺の味方に攻撃速度や回復を与えるため、配置そのものがシナジーの一部でした。 [\[tftactics.gg\]](https://tftactics.gg/set-update/)

Set 17 の N.O.V.A. は特定チャンピオンの Strike を選択してパワーサージ時に発動させるため、構成と配置・選択が絡むシナジーです。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 配置の上手さで差が出る
* 同じ構成でもプレイヤースキルが出る
* 対戦相手ごとに調整できる

**弱み**

* スカウトが必要
* Zephyr/Shroud 的な妨害に弱い
* 初心者には負担が大きい

## 13. 時間経過・スケーリング型

* Ascension 系

防御型タンクや回復型サポートと相性がよく、逆にバースト構成には弱いです。The Mighty Mech のような防御スタンス型や、Tahm Kench のような回復タンクは、このスケーリング型を支える前衛として機能します。 [\[mobalytics.gg\]](https://mobalytics.gg/blog/tft/set-11-guide-how-to-play-storyweaver/), [\[wiki.leagu...egends.com\]](https://wiki.leagueoflegends.com/en-us/TFT:Glutton)

**強み**

* 長期戦で非常に強い
* タンク・回復・シールドと相性がよい
* 終盤盤面で安定しやすい

**弱み**

* 初動バーストに弱い
* キャリーが早く落ちると何もできない
* 戦闘時間を稼げる前衛が必須

## 14. 状態異常・妨害・制御型

### 代表例

* Stun
* Taunt
* Charm
* Knock-up
* Shred / Sunder
* Disruptor
* Galio / Sejuani / Nautilus / Morgana 系

この型は、**敵に行動させない、火力を下げる、防御を削る**ことを目的にします。Galio の挑発・防御上昇・範囲ダメージ・味方シールドは、タンク型でありながら妨害型でもあります。 [\[tftactics.gg\]](https://tftactics.gg/champions/tahm_kench/), [\[op.gg\]](https://op.gg/tft)

Set 17 でも N.O.V.A. の Aatrox が敵に Shred/Sunder を与えるように、防御低下をチーム全体の火力補助として使う設計が見られます。 [\[metatft.com\]](https://www.metatft.com/units/Galio)

**強み**

* 格上相手にも勝ち筋を作れる
* 敵キャリーの初回スキルを遅らせられる
* 味方火力を間接的に伸ばせる

**弱み**

* CC無効・Unstoppable に弱い
* ダメージそのものは不足しがち
* 対象や範囲がズレると弱い

| 奇襲型 | 後衛を直接狙う          | Assassin、Rogue、Zed、Akali                          |
| 経済型 | ゴールド・XP・アイテムを増やす | Ahri神、Mercenary、Heartsteel、Oracle Tahm Kench      |

## 私見としての整理

TFTのSet設計は、完全に新しいものを毎回作っているというより、**既存の遊びの型を、テーマに合わせて言い換えている**ことが多いです。

たとえば、

* 「別キャラを食べる」  
  → **育成型 + リソース変換型**
* 「ずっと防御する」  
  → **要塞型 + 時間稼ぎ型**
* 「クエストを攻略する」  
  → **達成報酬型 + 経済/アイテム型**
* 「負けて報酬を貯める」  
  → **賭け型 + キャッシュアウト型**
* 「神を選んで供物を得る」  
  → **メタ進行型 + クエスト/経済型**

という感じです。

なので新Setを見るときは、チャンピオン名やシナジー名よりも、

1. **これは経済を増やす型か**
2. **戦闘中に強くなる型か**
3. **盤面外で進捗を貯める型か**
4. **1体を化け物にする型か**
5. **召喚体やコピーで盤面を増やす型か**
6. **配置で性能が変わる型か**
7. **負けを報酬に変える型か**

で見ると、かなり早く理解できます。
